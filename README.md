

### 📈 S&P 500 Trend Prediction Web App (LSTM)
**Python | TensorFlow | Streamlit | GitHub**

- Built an LSTM-based deep learning model to predict S&P 500 next-period trend (UP/DOWN).
- Engineered technical indicators (SMA, RSI) to enhance prediction performance.
- Implemented sequence modeling using 6-month rolling window.
- Deployed an interactive web application on Streamlit Cloud.
- Displayed prediction probabilities and live trend visualization dashboard.
- Integrated feature scaling and model persistence using Joblib.
